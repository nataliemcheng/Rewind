The Code for Rewind


                        // Get the string from the page
                        // controller function
                        function getString() {
                            // alert goes invisible at the start
                            document.getElementById("alert").classList.add("invisible"); 

                            let userString = document.getElementById("userString").value;
                            let revString = reverseString(userString);
                            displayString(revString);
                        }

                        // Reverse the string
                        // logic function
                        function reverseString(userString) {
                            let revString = [];

                            // reverse a string using a loop
                            for (let index = userString.length - 1; index >= 0; index--) {
                                revString += userString[index];
                            }

                            return revString;
                        }

                        // Display the reversed string to the user
                        // view function
                        function displayString(revString) {
                            // write to the page
                            document.getElementById("msg").innerHTML = `Your string reversed is: ${revString}`;
                            // show the alert box
                            document.getElementById("alert").classList.remove("invisible");
                        }

Rewind is structured in three functions:

getString()

The function getString() is the controller function. At the start, the alert of "Well Done!" with the reversed text is made invisible by taking the ID added to the alert, and then adding an "invisible" class onto the element. This "invisible" class will be taken away later. Next, two variable are declared: 1. The string the user entered (userString), and 2. The reversed string (revString). userString's value is done by finding the element with the appropriate ID using document.getElementById and then taking the value from that element. revString is the result of the function reverseString, with userString as its input. Lastly, the display function displayString() is called so the user can view their result.

reverseString()

The function reverseString() is the logic function. It takes a string parameter userString, which is the string that the user entered that they want to reverse. The function starts by instantiating an array called revString, which will be used to put the letters of the string in reversed order within the for loop. The for loop starts with the end of the string, taking each letter starting from the back and adding it to revString, which results in the reversed word. After each iteration of the loop, the index is decreased, so the loop makes its way to the front of word by the end. After the for loop is completed, the array is returned.

displayString()

The function displayString() is the view function, which controls how the user will see their reversed string. By using the same process from earlier of getting an element by its ID, the result can be input into the app with innerHTML. The function also removes the invisible class applied to the alert from earlier, so the user can see the alert on their page with their final reversed string.
